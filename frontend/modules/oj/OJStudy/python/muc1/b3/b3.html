🎯 Danh Sách (List) trong Python

1. Khai báo và Khởi tạo

List là một cấu trúc dữ liệu dùng để lưu trữ nhiều giá trị trong một biến duy nhất. Các phần tử được bao quanh bởi dấu ngoặc vuông []. 

# Khởi tạo list trống
my_list = []

# List chứa các số
numbers = [1, 2, 3, 4, 5]

# List chứa nhiều kiểu dữ liệu
mixed = ["Python", 10, True, 3.14]

2. Truy xuất phần tử (Index)

Python sử dụng index bắt đầu từ 0. Bạn cũng có thể dùng index âm để đếm ngược từ cuối. 

fruits = ["Táo", "Chuối", "Cam"]
print(fruits[0])  # Output: Táo
print(fruits[-1]) # Output: Cam (phần tử cuối)

3. Cắt danh sách (Slicing)

Cú pháp: list[start:end] (lấy từ start đến sát end). 

nums = [0, 1, 2, 3, 4, 5]
sub = nums[1:4] # Lấy index 1, 2, 3 -> [1, 2, 3]

4. Các thao tác phổ biến

  • append(x): Thêm x vào cuối list.
  • del list[i]: Xóa tại vị trí i.
  • pop(): Lấy và xóa phần tử cuối.
  • sort(): Sắp xếp list.
a = [3, 1, 2]
a.append(4)    # [3, 1, 2, 4]
a.sort()      # [1, 2, 3, 4]
print(len(a)) # Độ dài list: 4

5. List Comprehension (Nâng cao)

Cách tạo list mới siêu nhanh chỉ với 1 dòng code!

# Tạo list các số bình phương từ 0-4
squares = [x**2 for x in range(5)]
# Kết quả: [0, 1, 4, 9, 16]
