1. Python là gì?

Hãy tưởng tượng Python là một người bạn ngoại quốc rất thông minh nhưng chỉ hiểu những câu lệnh đơn giản, rõ ràng. Python hiện đang là ngôn ngữ lập trình phổ biến nhất thế giới, dùng trong trí tuệ nhân tạo (AI), làm web, phân tích dữ liệu, và tự động hoá.

Cú pháp tự nhiên

Mã nguồn Python gần giống tiếng Anh, không dùng quá nhiều dấu ngoặc phức tạp như C++ hay Java.

Thụt đầu dòng (Indentation)

Python dùng khoảng trắng (Space/Tab) ở đầu dòng để gom nhóm các lệnh. Thụt dòng sai = Code lỗi!

2. Đi Sâu Vào Nhập & Xuất Dữ Liệu

2.1 Xuất dữ liệu (Output) - Hàm `print()`

Lệnh print() dùng để in thông tin ra màn hình. Tuy nhiên, nó có rất nhiều "tuyệt chiêu":

A. F-string (Định dạng chuỗi cực xịn từ Python 3.6)

Chỉ cần thêm chữ f trước chuỗi, bạn có thể nhét thẳng biến vào trong cặp ngoặc nhọn {}. 

ten = "Huy"
tuoi = 20
print(f"Xin chào, mình là {ten}, năm nay mình {tuoi} tuổi.")

B. Tham số `end` và `sep`

  • sep: Ký tự phân cách giữa các biến (mặc định là dấu cách).
  • end: Ký tự cuối cùng sau khi in (mặc định là xuống dòng `\n`). 
# Thay đổi dấu phân cách
print("Cam", "Táo", "Nho", sep=" - ") # Kết quả: Cam - Táo - Nho

# Không xuống dòng sau khi in
print("Đang tải", end="...")
print("Xong!") # Kết quả: Đang tải...Xong!

2.2 Nhập dữ liệu (Input) và Ép Kiểu

Dùng input() để lấy dữ liệu từ bàn phím. Lưu ý cực kỳ quan trọng: Mọi thứ bạn nhập vào qua `input()` đều được Python hiểu là Chuỗi chữ (String).

Nếu bạn muốn tính toán toán học, bạn BẮT BUỘC phải "ép kiểu" (biến chữ thành số) bằng `int()` (số nguyên) hoặc `float()` (số thập phân). 

# Nhập chữ thông thường
ten = input("Tên bạn là gì? ")

# Ép kiểu sang số nguyên để tính toán
nam_sinh_chuoi = input("Nhập năm sinh: ")
nam_sinh_so = int(nam_sinh_chuoi)
tuoi = 2026 - nam_sinh_so
print(f"Tuổi của bạn là: {tuoi}")

# Viết gộp cho lẹ:
can_nang = float(input("Nhập cân nặng (kg): "))

3. Rẽ Nhánh Logic: If...Elif...Else

Giống như trong đời sống: "Nếu trời mưa thì ở nhà, ngược lại thì đi chơi".

Cấu trúc IF - ELIF - ELSE

Dùng elif (viết tắt của else if) khi có nhiều hơn 2 trường hợp. 

diem = 7.5

if diem >= 8.0:
    print("Học sinh Giỏi")
elif diem >= 6.5:
    print("Học sinh Khá")
else:
    print("Học sinh Trung bình")

Toán tử Logic (Kết hợp nhiều điều kiện)

  • and : Cả hai đều ĐÚNG
  • or : Chỉ cần 1 cái ĐÚNG
  • not : Đảo ngược (ĐÚNG thành SAI) 
tuoi = 20
co_ve = True

if tuoi >= 18 and co_ve:
    print("Được vào xem phim rạp")

4. Tổng hợp 10 Ví dụ Minh Hoạ Thực Tế

Đọc và nghiền ngẫm các đoạn code này để hiểu sâu cách kết hợp Nhập/Xuất và Điều kiện nhé!

Ví dụ 1 Nhập xuất cơ bản 
ten = input("Nhập tên: ")
print(f"Xin chào {ten}!")
Ví dụ 2 Ép kiểu để tính chu vi 
canh = int(input("Cạnh hình vuông: "))
print(f"Chu vi: {canh * 4}")
Ví dụ 3 Phép chia lấy dư (%) 
so = int(input("Nhập 1 số: "))
if so % 2 == 0:
    print("Số chẵn")
else:
    print("Số lẻ")
Ví dụ 4 If lồng nhau (Nested If) 
n = 10
if n > 0:
    if n % 2 == 0:
        print("Số dương chẵn")
Ví dụ 5 Dùng 'or' 
thang = 8
if thang == 7 or thang == 8:
    print("Đang là mùa hè!")
Ví dụ 6 So sánh chuỗi 
pass = input("Mật khẩu: ")
if pass == "12345":
    print("Mở khóa!")
else:
    print("Sai mật khẩu")
Ví dụ 7 If một dòng (Ternary) 
tuoi = 20
status = "Lớn" if tuoi >= 18 else "Nhỏ"
print(status)
Ví dụ 8 Kiểm tra số âm dương 
x = float(input("Nhập x: "))
if x > 0: print("Dương")
elif x 0: print("Âm")
else: print("Bằng Không")
Ví dụ 9 Kết hợp and & or 
toan = 9
van = 8
if (toan >= 8 and van >= 8) or toan == 10:
    print("Thưởng lớn!")
Ví dụ 10 Đổi nhiệt độ C -> F 
c = float(input("Độ C: "))
f = (c * 9/5) + 32
print(f"{c} độ C = {f} độ F")
