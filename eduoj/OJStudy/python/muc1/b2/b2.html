1. Thụt lề (Indentation) - Luật sắt trong Python

Khác với C++ hay Java dùng dấu ngoặc nhọn {}, Python dùng khoảng trắng để xác định các khối lệnh. Thông thường là 4 dấu cách.

Cảnh báo lỗi!

Nếu bạn thụt lề sai, Python sẽ báo lỗi IndentationError ngay lập tức.

VÍ DỤ 
# Đúng ✅
if True:
    print("Chào Python!")

# Sai ❌ (Lỗi IndentationError)
if True:
print("Chào Python!")

2. Vòng lặp while - Lặp cho tới khi chán

Vòng lặp while sẽ chạy liên tục khối lệnh bên trong chừng nào điều kiện vẫn còn Đúng (True). 

while điều_kiện:
    # Thực hiện lệnh
else:
    # Chạy khi điều kiện sai
  • Dùng khi không biết trước số lần lặp.
  • Cẩn thận với lặp vô tận (Infinity Loop)!

3. Vòng lặp for...in - Duyệt qua mọi thứ

Trong Python, for cực kỳ mạnh mẽ để duyệt qua các phần tử của một danh sách (list), chuỗi (string) hoặc một phạm vi (range).

DUYỆT CHUỖI 
for x in "PYTHON":
    print(x)
HÀM RANGE() 
for i in range(5):
    print(i) # 0 -> 4
SO VỚI C/C++

Python viết ngắn gọn hơn nhiều, không cần i++, i < n...

4. Lệnh điều hướng: break & continue

break

"Dừng cuộc chơi!" - Thoát khỏi vòng lặp ngay lập tức dù điều kiện vẫn đúng.

continue

"Nhảy cóc!" - Bỏ qua các lệnh còn lại trong lượt lặp này và bắt đầu lượt mới.

Ghi chú Pythonic (Viết code như chuyên gia)

  • Không cần Switch-case: Python sử dụng if-elif-else (Từ bản 3.10 có match-case nhưng if vẫn phổ biến nhất).
  • Boolean: Hãy viết while True: thay vì while 1:.
  • Lặp thông minh: Tránh dùng range(len(my_list)), hãy dùng trực tiếp for item in my_list:.
